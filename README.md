# 📊 Retail Sales Analysis Dashboard | Power BI Project

## 📌 Project Overview
This project analyzes retail sales data using Power BI to uncover valuable business insights. The interactive dashboard helps stakeholders monitor sales performance, profitability, customer behavior, and regional trends for data-driven decision making.

---

## 🎯 Project Objectives

- Analyze overall sales performance
- Identify high-performing products and categories
- Monitor profit across different regions
- Understand customer purchasing patterns
- Compare sales over time
- Build an interactive dashboard for business users

---

## 🛠️ Tools Used

- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Excel Dataset

---

## 📈 Dashboard Features

- KPI Cards
- Sales Trend Analysis
- Regional Sales Map
- Category-wise Sales
- Top Products
- Profit Analysis
- Interactive Filters (Slicers)
- Dynamic Charts

---

## 📊 Key Business Insights

✔ Overall sales performance

✔ Best-performing product categories

✔ Most profitable regions

✔ Monthly sales trends

✔ Customer purchasing behavior

✔ Products generating the highest revenue
